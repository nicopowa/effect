class Effect
lightweight animation engine / animate HTML elements CSS props
Nico Pr 2020
constructor(target:Element, frames:number, props:Object, ease:string=, delay:number=, override:boolean=, callback:Function=):Effect
target : who wants to move ?
frames : number of frames
props : CSS props
ease : easing function name
delay : frames before start
override : stop current effect
callback : effect end function
return : Effect instance
